# Legal Checklist

Payments, public API access, and paid subscriptions stay disabled until this checklist is complete.

## SAM 3D Objects

- Add SAM License notice to `LEGAL.md`.
- Add SAM License notice to the app legal page.
- Add SAM License notice to internal deployment docs.
- Confirm obligations for any distributed SAM Materials or derivatives.
- Set `LEGAL_REVIEW_SAM_LICENSE=true` only after review.

## Product terms

- State that users retain uploaded content.
- State that, as between the user and CappyMesh, users own generated asset outputs subject to source rights, applicable law, and third-party rights.
- Prohibit reverse engineering of SAM Materials or underlying model components.
- Add warranty disclaimer and as-is output language.
- Clarify that users are responsible for rights, quality, safety, and suitability checks.

## Privacy

- Treat uploaded photos as personal data.
- Ship asset and upload deletion.
- Document retention windows.
- Avoid training on uploads without explicit opt-in.
- Add subprocessors before launch.

## Prohibited uses

- ITAR-controlled activity.
- Military or warfare purposes.
- Nuclear applications.
- Espionage.
- Guns or illegal weapons.
- Sanctions or export-control violations.
- Celebrity likeness abuse.
- Third-party IP misuse.
